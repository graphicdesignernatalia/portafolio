# Natalia Farías | Portfolio

Portfolio profesional desarrollado para presentar mi experiencia en periodismo, diseño gráfico y comunicación digital.

El sitio reúne proyectos de branding, UX/UI, diseño editorial, diseño web y trabajos realizados para medios de comunicación como TVN y Radio Bío Bío.

## Tecnologías

- React
- TypeScript
- Vite
- Tailwind CSS 4
- Lucide React

## Características

- Diseño completamente responsive.
- Animaciones suaves y microinteracciones.
- Portafolio organizado por proyectos.
- Modal interactivo para visualizar cada trabajo.
- Descarga de CV.
- Optimización para SEO.
- Desarrollado con enfoque en rendimiento y accesibilidad.

## Demo

🔗 https://natalia-farias.vercel.app

## Instalación

Clona el proyecto:

```bash
git clone https://github.com/graphicdesignernatalia/portfolio.git
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el proyecto:

```bash
npm run dev
```

Compila para producción:

```bash
npm run build
```
